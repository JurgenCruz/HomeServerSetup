# Configure ZFS

[![en](https://img.shields.io/badge/lang-en-blue.svg)](Configure%20zfs.md)
[![es](https://img.shields.io/badge/lang-es-blue.svg)](Configure%20zfs.es.md)

We will use ZFS for our storage needs. Please select the option that matches your situation. If it is a first installation, select `Create ZFS pool`. If it is a reinstallation, select `Import existing ZFS pool`. If in the future you need to do a hard drive migration for your pool, select `Migrate ZFS pool drives`.
